<a target="_blank" href="?conf=sappId=addd-prel-curveXX,theorionId=proof,aspectId=curve,subessayId=frenet-serret">
is continuously differentiable,
</a>

then
    <div style="text-align:center;">
        ds / ¦AR¦r¦¦ = |d𝗿| / ¦AR¦r¦¦ = vdq / ¦AR¦r¦¦. &nbsp;&nbsp;&nbsp;(*)
    </div>
<br>

Returning from <><>CXX2E, let q be such a curve parameter that

    <div style="text-align:center;">
        (¦AD¦𝘃∆q¦¦ - ¦AB¦∆𝗿¦¦) // to constant vector
        ¦imageOfR,imageOfD¦𝗵 ¦¦ = ¦imageOfR,imageOfD¦'rd'¦¦.
    </div>
This is equivalent to,

    <div style="text-align:center;">
        (¦AD¦𝘃∆q¦¦ - ¦AB¦∆𝗿¦¦) ⊺ ¦rd¦𝗵 ¦¦ = 0, and ∆q = ∆𝗿⊺𝗵 / 𝘃⊺𝗵.
    </div>
Using
    <div style="text-align:center;">
        ∆𝗿 = 𝘃∆q + 𝗼(∆q)
    </div>
implies ultimate similarity of underlined triangles. For example, ratios to r
will be:
    <div style="text-align:center;">
        ¦AB¦∆𝗿¦¦ / ¦AR¦r¦¦ = ¦AD¦𝘃∆q¦¦ / ¦AR¦r¦¦ + 𝗼(∆q), <br>

        ¦AB¦|∆𝗿|¦¦ / ¦AR¦r¦¦ + o<sub>2</sub>(∆q) =<br>
        ¦AD¦v∆q¦¦ / ¦AR¦r¦¦ + o<sub>3</sub>(∆q). &nbsp;&nbsp;&nbsp;(**)
    </div>
which is a finite form of above differential equalities (*).
